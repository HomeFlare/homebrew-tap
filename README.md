# HomeFlare Homebrew Tap

Homebrew formulae for [The HomeFlare Project](https://github.com/HomeFlare).

## Install

```bash
brew tap homeflare/tap
brew install homeflare
```

## Usage

```bash
homeflare secrets set KEY "value" --private
homeflare secrets get KEY
homeflare secrets ls
homeflare secrets deploy

# Shortcut alias
hfs set KEY "value"
```

## Upgrade

```bash
brew upgrade homeflare
```
